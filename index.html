Herramienta educativa para trabajar con Álgebra Relacional

Pireal es una herramienta educativa para su uso en la cátedra Base de Datos, permite al usuario experimentar de forma interactiva con Álgebra Relacional.

Con Pireal podrá crear bases de datos, agregar tablas y hacer consultas en el lenguaje de Álgebra Relacional.

Programadores!

  • Pireal está desarrollado completamente en Python.

  • El código fuente está disponible en GitHub.

Cómo colaborar con el proyecto?

  • Podés reportar algún error o sugerir una nueva característica acá.

  • Pireal esta disponible en Inglés y Español. Querés traducir Pireal a otro idioma? Hacélo!.
