Herramienta para trabajar con Álgebra Relacional

Pireal es una herramienta que permite evaluar consultas escritas en el lenguaje de Álgebra Relacional y visualizar el resultado de una consulta en particular.

Las consultas se hacen a una base de datos. Una base de datos contiene tablas/relaciones, tanto la base de datos como las relaciones, pueden ser creadas/editadas desde la herramienta o desde un editor de texto.

Creación de la base de datos

Una base de datos de Pireal es un archivo .pdb, para crearlo tenés dos opciones, desde la UI o creando un archivo:

El archivo debe tener extensión .pdb con la siguiente estructura:

  • Ejemplo del archivo database.pdb:

    • @personas:idPersona/numeric,nombrePersona/char,fecha_nac/date
    11,Gabriel,20/01/1991
    22,Rodrigo,19/06/2000
    33,Mariela,18/11/71

    @aptitudes:idPersona/numeric,aptitud/char
    22,Gamer
    11,Python
    33,Chef

    El header (nombres de campos) comienza con @seguido del nombre de la relación o tabla, luego : (dos puntos) y a continuación el nombre de cada campo separados por comas, a su vez, cada campo tiene asociado un tipo separado por /, el tipo puede ser numérico (numeric), alfanumérico (char) o fecha/hora (date). En las siguientes líneas se especifican las tuplas, es decir, los valores para cada campo, separadas por comas.

  • Guardas el archivo y ahora podés cargar la base de datos desde Pireal.

Consultas

Una vez que tengas cargada una base de datos, podés hacer consultas como éstas:

consulta_1 = personas njoin aptitudes
consulta_2 = select idPersona == 11 (consulta_1)
consulta_3 = project nombrePersona, aptitud (consulta_2)

O simplemente:

project nombrePersona, aptitud (select idPersona == 11 (personas njoin aptitudes))

Si querés agregar comentarios a una consulta, usá la siguiente sintáxis:

-- Ésto es un comentario
q1 = select idPersona == 2
-- Otro comentario
q2 = project nombrePersona (q1)

Los archivos de consultas, tienen extensión .pqf, por lo que podés crear consultas editando un archivo y luego abrir esa consulta desde Pireal.
