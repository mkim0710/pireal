Relational Algebra Query Evaluator

Pireal es una herramienta que permite evaluar consultas escritas en el lenguaje de Álgebra Relacional y visualizar el resultado de una consulta en particular, todo desde una interfáz muy intuitiva.

Las consultas se hacen a una base de datos. Una base de datos contiene tablas/relaciones, tanto la base de datos como las relaciones, pueden ser creadas/editadas desde Pireal o desde un editor de texto.

    • Se está trabajando en el empaquetado para Debian/Ubuntu, mientras tanto puedes usar el tarball.

  • Windows

    • Descarga el instalador: pireal-1.0-installer.exe.

  • Desde el código fuente

    • Python 3

    • PyQt5

    Luego:

    • Descarga el tarball multiplataforma: pireal-1.0.tar.gz.

    • Descomprimir, entrar a la carpeta e instalar:

      • tar -xvf pireal-1.0.tar.gz
      cd pireal-1.0
      sudo python setup.py install

Creación de la base de datos

Una base de datos de Pireal es un archivo .pdb, para crearlo hay dos opciones, desde la UI o creando un archivo:

El archivo debe tener extensión .pdb con la siguiente estructura:

  • Ejemplo del archivo database.pdb:

    • @personas:idPersona,nombrePersona,fecha_nac
    11,Gabriel,20/01/1991
    22,Rodrigo,19/06/2000
    33,Mariela,18/11/1976
    44,Mercedes,06/09/1994

    @aptitudes:idPersona,aptitud
    44,Rocas
    22,Gamer
    11,Python
    33,Chef
Nota: No debe existir espacios en los nombres de campo después de la coma.

Consultas

Una vez que tengas cargada una base de datos, se pueden hacer consultas como éstas:

consulta_1 = personas njoin aptitudes
consulta_2 = select idPersona == 11 (consulta_1)
consulta_3 = project nombrePersona, aptitud (consulta_2)

O simplemente:

project nombrePersona, aptitud (select idPersona == 11 (personas njoin aptitudes))

También se puede comentar las consultas:

-- Ésto es un comentario
q1 = select idPersona == 2
-- Otro comentario
q2 = project nombrePersona (q1)
